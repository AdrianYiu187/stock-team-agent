## 2. v5.21 設計 — Hybrid TTL + Live Recompute

### 2.1 核心原則

| 區段 | 策略 | 理由 |
|------|------|------|
| `tickers` | 靜態（11 ticker list） | Ticker universe 變更需顯式升級 |
| `fundamentals` | **Live yfinance + 24h TTL cache** | PE/ROE/PEG/growth 變動慢（季度），24h 內重複抓是浪費 |
| `v5_10_scores` | **Live recompute** from cached fundamentals | Scoring algorithm 改了不需重抓 JSON |
| `v5_11_3_scores` | **Live recompute** from cached fundamentals | 同上 |
| `std_quant` | **Live recompute** from live scores | Quant 純函數，無需 cache |

### 2.3 TTL Cache 設計（fixture_cache.py）

```python
"""
v5.21: TTL-based fixture cache for yfinance fundamentals.

Cache key: ticker (e.g. "AAPL")
Cache value: {pe, roe, peg, growth, fetched_at, source}
TTL: 24h (configurable via FIXTURE_TTL_HOURS env var)
Storage: scripts/tests/fixtures/.cache/fundamentals/<TICKER>.json
"""

DEFAULT_TTL_HOURS = 24
CACHE_DIR = Path(__file__).parent.parent / "tests" / "fixtures" / ".cache" / "fundamentals"

def get_fundamentals(ticker: str, *, force_refresh: bool = False) -> dict:
    """
    Returns: {pe, roe, peg, growth, fetched_at, source}
    Side effect: writes cache file if miss or stale
    Falls back to stale cache if yfinance raises (offline-safe)
    """
    cached = _read_cache(ticker)
    if cached and not _is_stale(cached, ttl_hours=DEFAULT_TTL_HOURS) and not force_refresh:
        return cached
    try:
        fresh = _fetch_from_yfinance(ticker)
        _write_cache(ticker, fresh)
        return fresh
    except Exception as e:
        if cached:
            log_warning(f"yfinance failed for {ticker}, using stale cache: {e}")
            return cached
        raise  # No cache, can't proceed
```

### 2.4 Live Score Recompute（live_score_engine.py）

```python
"""
v5.21: Re-executes v5.10 and v5.11.3 scoring from fundamentals.
Replaces hardcoded v5_10_scores and v5_11_3_scores sections.
"""

def recompute_all_scores(tickers: list[str]) -> dict:
    """
    Returns: {
        "v5_10_scores": {ticker: score, ...},
        "v5_11_3_scores": {ticker: score, ...},
        "std_quant": {ticker: quantile_bucket, ...}
    }
    """
    fundamentals = {t: get_fundamentals(t) for t in tickers}
    return {
        "v5_10_scores": _v5_10_scoring(fundamentals),
        "v5_11_3_scores": _v5_11_3_scoring(fundamentals),
        "std_quant": _quant_bucketing(fundamentals),
    }
```

### 2.5 Backward Compatibility
