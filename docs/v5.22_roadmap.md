## 2. Pitfall 候選 (1 個, P41)

### P41 — tech.ma50_factor ratio → absolute diff

**Problem**:
- `price/ma50` ratio 在 ma50<<price 範圍梯度極小
- 真實場景復現: `price=$200, rsi=50, macd=0, mom=0.05` → ma50=50,100,150 → score=`0.6234` (完全相同)

**Fix 設計**:
```python
# 改 absolute diff-based
ma50_pct = (price - ma50) / ma50  # 真實相對差 [-1, +∞]
if ma50_pct >= 0.20:        # price > ma50 by 20%+
    ma50_factor = min(0.85, 0.6 + 0.2 * (ma50_pct - 0.20) / 0.20)  # 0.6..0.85
elif ma50_pct >= -0.20:     # ±20% neutral band
    ma50_factor = 0.4 + 0.2 * (ma50_pct + 0.20) / 0.40  # 0.4..0.6
else:                       # price < ma50 by 20%+
    ma50_factor = max(0.2, 0.4 - 0.2 * (-ma50_pct - 0.20) / 0.20)  # 0.2..0.4
```
